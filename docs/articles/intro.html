Getting started with DiscordRPC C#

Getting started with DiscordRPC-C# is easy! With automatic nuget packages and a fully fledge Unity3D package, you can get started in no time.

Dependencies

  • Newtonsoft.JSON
  • .NET 3.5+ or Unity3D 5.5+

Select Project Type

Standard .NET Project - Recommended. For most users that are making a .NET Application

Unity3D Project - For Unity3D users that want to add Rich Presence to their game.

MonoGame Project / Modding - For developers using custom MonoGame engines or modding existing games.
