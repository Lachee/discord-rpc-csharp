Avatar

The avatar hash of the user. Too get a URL for the avatar, use the GetAvatarURL(User.AvatarFormat, User.AvatarSize). This can be null if the user has no avatar. The GetAvatarURL(User.AvatarFormat, User.AvatarSize) will account for this and return the discord default.

GetAvatarURL(User.AvatarFormat, User.AvatarSize)

Gets a URL that can be used to download the user's avatar. If the user has not yet set their avatar, it will return the default one that discord is using. The default avatar only supports the PNG format.
